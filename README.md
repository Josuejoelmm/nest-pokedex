# Run in development

1. Clone the repository
2. Run

```
npm i
```

3. Have Nest CLI installed

```
npm i -g @nestjs/cli
```

4. Set up Database

```
docker-compose up -d
```

5. Rebuild Data base with the Seed

```
http://localhost:3000/api/v2/seed
```

## Stack used

- MongoDB
- NestJS
